#User Story Specification: Visualizar datos de plan

##1.	Description

Como usuario quiero poder visualizar los datos de un determinado plan.

##2.	Acceptance Criteria

- Se debe poder ver cada uno de los datos del usuario.
- Se debe poder ver todos los viajes hechos.
- Se debe poder agregar viajes.
- Se debe poder acceder al total de viajes.
- Si no hay viajes, o no se puede acceder a estos, mostrar el respectivo error.
- Los datos a visualizar son los mismos que el usuario ingresa al dar de alta el viaje.
