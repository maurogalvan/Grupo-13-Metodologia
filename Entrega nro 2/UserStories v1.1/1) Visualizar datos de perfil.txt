#User Story Specification: Visualizar datos de perfil.

##1.	Description


Como usuario quiero poder visualizar mis datos personales y de perfil para así poder ver mis datos y mis viajes.

##2.	Acceptance Criteria

- Se debe poder ver cada uno de los datos del usuario:
	- Nombre.
	- Apellido.
	- Mail.
	- Pais.
	- Fecha de nacimiento.
	- cantidad de paises visitados
	- lista de paises visitados
	- cantidad de Km recorridos
	- Estado de premium
	- Estado notificaciones
- Se debe poder ver todos los viajes hechos.
- Se debe poder acceder al total de viajes.
- Si no hay viajes, o no se puede acceder a estos, mostrar el respectivo error.
