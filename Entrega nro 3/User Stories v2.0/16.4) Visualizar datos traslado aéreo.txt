#User Story Specification: Visualizar datos traslado aéreo.

##1.	Description

Como usuario quiero poder visualizar los datos de mi vuelo o traslado aéreo para tener una mejor organización.

##2.	Acceptance Criteria

- Se debe poder visualizar la compañía encargada de nuestro traslado.
- Se debe poder visualizar fecha inicio y fin del traslado.
- se debe poder ver el tipo y la ubicación.
- Se debe poder visualizar el asiento que nos corresponde.
- Se debe poder ver un tiempo estimado de la duración de vuelo.
- Se debe poder visualizar el número de vuelo que corresponde al traslado.
- Se deben poder visualizar datos del aeropuerto de llegada y salida.
- Se debe poder visualizar información de la aeronave que nos traslada.
